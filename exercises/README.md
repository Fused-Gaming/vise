# VISE Exercises & Practical Challenges

This directory contains hands-on exercises for each module. Complete these to reinforce your learning and prepare for module assessments.

## Exercise Structure

Each module has dedicated exercises organized by difficulty:
- **Foundation**: Basic exercises to build core skills
- **Intermediate**: Apply concepts to realistic scenarios
- **Advanced**: Challenge exercises for deeper understanding
- **Integration**: Combine multiple concepts

## How to Use These Exercises

1. **Work Through Sequentially**: Start with foundation exercises
2. **Hands-On Practice**: Don't just read - actually code
3. **Test Your Work**: All exercises should include tests
4. **Seek Feedback**: Share your solutions for peer review
5. **Iterate**: Improve based on feedback

## Submission

- Create a GitHub repository for your exercise solutions
- Organize by module and exercise number
- Include README for each exercise with your approach
- Add tests where applicable
- Request peer review in Discord

## Getting Help

- VISE Discord channels for each module
- Office hours with mentors
- Peer study groups
- Exercise walkthroughs (after attempts)

## Grading

Exercises are not graded but are essential preparation for:
- Module assessments
- Sprint projects
- Token issuance requirements

Quality exercise completion correlates strongly with assessment success!
